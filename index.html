Project 1: To Do list

Project Overview

The To-Do List CLI Application is a simple yet functional task management system built using Python. This command-line interface (CLI) application allows users to add, view, and mark tasks as completed, helping them stay organized and track their progress efficiently.

Features

Add a Task: Users can enter tasks to be completed.

View Tasks: Displays a numbered list of all pending tasks.

Mark Task as Done: Users can mark tasks as completed, moving them to a separate "Done" list.

View Completed Tasks: Users can review the tasks they have already finished.

Simple and Interactive Menu: A user-friendly menu system guides users through the app.

How It Works

The application starts with a menu, allowing users to choose an action.

Users can add tasks by typing them into the system.

When choosing to view tasks, the program lists all pending tasks.

Users can mark a task as done by entering the task number. The completed task is removed from the pending list and added to the "Done" list.

Users can also view their completed tasks at any time.

Technology Used

Programming Language: Python

Concepts Used: Lists, Functions, Loops, Conditional Statements

Future Enhancements

Implementing task deadlines and priorities

Adding data persistence using files or a database

Creating a GUI version using Tkinter or a web version using Django

Project 2: Youtube video downloader

Project Overview

The YouTube Video Downloader is a Python-based command-line application that allows users to download videos from YouTube using the yt-dlp library. This tool provides an easy and efficient way to save YouTube videos locally in the best available quality.

Features

Download Videos in Best Quality: The script automatically selects the highest video and audio quality available.

Custom Save Location: Videos are saved in a designated Downloads folder with proper filenames.

Single Video Download: The script ensures that only individual videos are downloaded (no playlists).

Progress Indication: Users can see the download status while the video is being saved.

Multiple Downloads: Users can input multiple video links in one session.

Error Handling: Provides user-friendly error messages if the download fails.

How It Works

The user enters the YouTube video URL in the terminal.

The script downloads the video in the best available quality.

The video is saved locally in the Downloads folder.

The user can enter another URL or type 'exit' to quit the program.

Technology Used

Programming Language: Python

Libraries: yt-dlp for video downloading

Future Enhancements

MP3 Audio Extraction: Option to download only the audio as an MP3 file.

Playlist Support: Ability to download entire YouTube playlists.

GUI Version: Implement a user-friendly graphical interface using Tkinter or PyQt.

URL Validation: Check if the provided link is a valid YouTube URL before downloading.
